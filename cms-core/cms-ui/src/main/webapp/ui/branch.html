Branch

git clone git@github.scm.corp.ebay.com:cloud-cms/cms.open.git to see the scripts behind all the examples on this site.

Query branches

Use following query to list all branches in CMS. 

GET /repositories/raptor-paas/branches

Query given branches

Use following query to list given branches in CMS.

GET /repositories/raptor-paas/branches/main

Create a branch

Create a branch in CMS.

POST /repositories/reptor-paas/branches
			
{
  "name" : "branch_11",
  "branchtype" : "sub",
  "parent_branch" : "c84feba0-8ef8-11e1-b0c4-0800200c9a66",
  "transaction" : "yes"
}

Commit a branch

Commit a branch in CMS.

POST /repositories/reptor-paas/branches/main
			
{
  "name" : "branch_11",
  "branchtype" : "sub",
  "parent_branch" : "c84feba0-8ef8-11e1-b0c4-0800200c9a66",
  "transaction" : "yes"
}

Abort a branch

Abort a branch in CMS.

DELETE /repositories/raptor-paas/branches/main
